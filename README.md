# AboveTheLaw – Los Santos Red Corrupt Cop Mod

A RAGE Plugin Hook mod for GTA V with Los Santos RED integration.

## Features
- Pull over vehicles with E (in car)
- Order driver out with E (on foot)
- Cuff suspects with T (real police animation)
- Secure in backseat with Y (they stay forever)
- Book suspects at jail locations with H (5 stations including BCSO Sandy Shores)

## Installation
1. Install RAGE Plugin Hook, Script Hook V, Script Hook V .NET (nightly), and Los Santos RED.
2. Copy `AboveTheLaw.dll` to `GTA V/plugins/AboveTheLaw/`.
3. Copy `AboveTheLaw_Settings.xml` to `GTA V/plugins/AboveTheLaw/XML/`.
4. Copy All xmls in the `AboveTheLawXMLConfiguration` to `plugins/LosSantosRED/` folder.

## Default Keys
- E = Pull Over / Order Out
- T = Cuff / Uncuff
- Y = Backseat In/Out
- F3 = Menu (Let Go)
- H = Jail Drop-off

Edit `AboveTheLaw_Settings.xml` to change keys.

## Booking Locations
- BCSO Sandy Shores
- Paleto Bay SO
- La Mesa PD
- Mission Row PD
- Los Santos SO

## Mp Maps need to be enabled in Los Santos Red

Look for the blue cylinder marker at each station.
